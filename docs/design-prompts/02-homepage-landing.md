# Prompt 02 — Homepage / landing

**Set:** A (existing site, redesigned)
**Status:** Live at `GET /` (`public/templates/index.html`).

## Prompt (copy-paste this into Claude Design)

```
Torrorèndum is a Catalan voting competition for Torrons Vicens torrons (heritage since 1775, warm brown/gold/cream palette, burgundy competition accent, heavy sans + warm serif type pairing). Design the landing page. It needs to communicate in one screen: this is a real competition with two phases — three weeks of open voting, then a knockout bracket — not an endless survey, and there's a countdown to the Gran Final reveal on Reis (Jan 6). Include: a bold Catalan headline ("Quin és el millor torró?"), a live countdown widget, a "how it works" strip (open season → bracket → gran final), and one strong call to action to start voting. Hint at the bracket structure visually so it reads as a competition with stakes. Avoid stock-photo Christmas imagery — use clean, neutral-background product photography consistent with Vicens' own shots.
```
