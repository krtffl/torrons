# Prompt 07 — Voting history

**Set:** A (existing site, redesigned)
**Status:** Live at `GET /history` (`public/templates/history.html`).

## Prompt (copy-paste this into Claude Design)

```
Torrorèndum: Catalan voting competition for Torrons Vicens, warm brown/gold/cream palette, burgundy accent. Design a chronological voting-history page: each entry shows the two torrons that faced off, which one the user picked (clearly marked winner vs. dimmed loser), the category, and a timestamp, with filter chips by category. Keep entries compact and scannable — this is a log people skim, not a page they read start to finish — and give it a distinct, quieter visual register than the high-energy duel screen it's a record of.
```
