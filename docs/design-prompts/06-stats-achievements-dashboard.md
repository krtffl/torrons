# Prompt 06 — Stats & achievements dashboard

**Set:** A (existing site, redesigned)
**Status:** Live at `GET /stats` (`public/templates/stats.html`).

## Prompt (copy-paste this into Claude Design)

```
Torrorèndum: Catalan voting competition for Torrons Vicens, warm brown/gold/cream palette, burgundy accent. Design a personal stats page: summary cards (total votes, categories unlocked, current streak), a per-category progress list (votes so far vs. minimum needed to unlock results, with a locked/unlocked state), and an achievement badge grid (milestone badges for 1, 5, 10, 25, 50, 100 votes — locked badges shown greyscale/dim, unlocked ones with a warm gold treatment). This page should feel rewarding to check mid-campaign, like a personal trophy case, not a raw analytics dump.
```
