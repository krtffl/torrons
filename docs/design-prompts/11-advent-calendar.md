# Prompt 11 — Advent calendar

**Set:** B (new screens for shipped features)
**Status:** Live at `GET /advent` (`public/templates/advent.html`). Shipped 2026-07-03.

## Prompt (copy-paste this into Claude Design)

```
Torrorèndum: Catalan voting competition for Torrons Vicens, warm brown/gold/cream palette, burgundy accent. Design an advent-calendar style grid for December 1–24: one "door" per day, each unlocking a themed duel-of-the-day (e.g. "el duel dels clàssics," "xocolata vs torró"). Past days show a small result/summary once opened; today's door is visually emphasized and inviting; future days are locked but visible, building anticipation rather than hiding entirely. Avoid literal snowflake/holly Christmas-calendar clichés — express "advent" through warm, toasted, gold-foil-adjacent materials consistent with the rest of the brand system, not through generic winter iconography.
```
