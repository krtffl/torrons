# Prompt 15 — Shareable result card

**Set:** B (new screens for shipped features)
**Status:** Live (deliberately plain v1) at `GET /share/card.png` (`internal/sharecard` package). Shipped 2026-07-03. Uses a bitmap font with no real typeface and a hand-rolled accent-stripping fallback for Catalan diacritics — of everything in Set B, this is the single highest-leverage prompt to commission, since Wrapped (Prompt 16) and the press kit (Prompt 18) both reuse its rendering engine and will inherit whatever visual pass this gets.

## Prompt (copy-paste this into Claude Design)

```
Torrorèndum: Catalan voting competition for Torrons Vicens, warm brown/gold/cream palette, burgundy accent. Design a downloadable vertical image, 1080×1920 (Instagram/WhatsApp Story format), summarizing a user's result — top torró, persona line, and the Torrorèndum mark, plus a short link or QR code back to the site. This single asset gets reused for three different moments (the personality reveal, the year-end Wrapped in Prompt 16, and the press kit in Prompt 18), so design it as a flexible template with 2–3 content variants, not one fixed layout. It's the single most important shareable asset in the product — it needs to look complete and desirable entirely on its own, the way a Spotify Wrapped card does, never like a cropped screenshot of a webpage.
```
